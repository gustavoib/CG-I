# Computação Grafica I
Repositório para trabalhos da disciplina de Computação Gráfica I (UFC)

## Usamos:
- OpenGL3 (GLFW + Glad) para gerar a cena
- imgui para menu

## Implementações
- Esfera
- Cilindro
- Cone
- Malha (usada aqui para gerar cubo)
- Plano
- Iluminação de Phong
- Luz spot
- Câmera
- Picking
- Transformações (translação, escala, rotação e cisalhamento)
